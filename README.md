 alu_regex-data-extraction_mmuuse-dot





This project is a Python-based tool that uses Regular Expressions (regex) to extract specific types of information from large blocks of text, such as API responses or web data.

The script extracts the following data types:

Email addresses

URLs

Phone numbers

Credit card numbers

Time (12-hour and 24-hour formats)

HTML tags

Hashtags

Currency amounts

Python 3.x

Project Files
Src (main.py) — Main Python script that performs the extraction.


input (input raw.txt) — Sample input file with unstructured data.



output (sample output.json) — Sample input file with structured data.




README.md — Project overview and usage instructions.



How to Use



1. Make sure Python 3 is installed on your system.


2. Run the script:

 
        python3 src/main.py
